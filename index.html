📖 ขั้นตอนตั้งค่า LINE Notify
1
เปิด notify.line.me แล้วล็อกอินด้วย LINE
2
กด "Generate token" → ตั้งชื่อ เช่น "ร้านซ่อมปริ้น"
3
เลือก Chat ที่รับแจ้งเตือน (ตัวเอง หรือ Group LINE ร้าน)
4
กด Generate → คัดลอก Token ทั้งหมด
5
เปิด Apps Script → ใส่ Token ในบรรทัด const LINE_TOKEN = "..."
6
เพิ่ม @LINE Notify เป็นเพื่อนใน LINE (สำคัญมาก!)
